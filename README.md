# HomeEase
## Script Location: homeease/scripts/homeease.py
## Execution Command: python3 homeease/scripts/homeease.py


## Run Instructions (GitHub Codespaces) ##

1. Open the HomeEase GitHub repository.
2. Click on Code → Codespaces → New codespace.
3. Wait for the environment to initialize (Python 3.12.1 and required dependencies will be available automatically).
4. Open the terminal in Codespaces.
5. Run the HomeEase application: type “python3 homeease/scripts/homeease.py”
